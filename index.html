About Me

I'm a PHP Developer working at Synalloy. On a daily basis, you can find me collaborating with teammates on technical problems, participating in code review, or writing modular reusable front/backend code. I enjoy thinking about problems and working together to overcome obstacles.

I've been programming and building websites since highschool. I have always wanted to be a Web Developer. Eventually I hope to lead a small team of devs.

My work experience includes Windows and Linux System Administration, SAP development, automation with PowerShell, and web development. At my current job we use a variety of different software that includes Slim Framework, React JS, and SQL Server. As a developer I constantly need to learn new things from frameworks and technologies to machine learning algorithms. In my off hours, I enjoy hiking, eating Mexican food and spending time with my beautiful fiancé.
